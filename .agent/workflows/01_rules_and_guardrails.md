---
description: 레포지토리 내 규칙을 명확히 고정하고, 에이전트 또는 개발자가 규칙을 벗어나지 않도록 가드레일을 설정한다.
---

- Trigger
  - 프로젝트 시작 직후
  - 규칙 위반이 반복될 때

- Goal
  - 모든 규칙을 실행 가능한 체크리스트로 고정한다

- Inputs
  - RULES.md
  - gemini 관련 규칙 문서
  - 코딩 컨벤션 문서

- Steps
  1. 모든 규칙 문서를 스캔한다
  2. 핵심 규칙을 최대 10개로 요약한다
  3. 체크리스트로 변환한다
     - 테스트 없이 코드 수정 금지
     - UI 스타일 토큰 외 사용 금지
     - 기능 변경 후 즉시 실행 검증
     - 대량 리팩터링 금지
  4. 자주 위반되는 패턴 Top 5를 정리한다

- Validation
  - 모든 변경 사항이 체크리스트를 통과했는지 기록된다
