# 슈퍼파워 사용하기 (스킬 체크)

**목표**: 모든 작업에 올바른 스킬과 워크플로우가 적용되도록 보장합니다.

## 단계

1.  **요청 분석**: 사용자의 요청을 분석합니다.
2.  **스킬 스캔**: `.agent/skills/skills_index.json` 또는 `.agent/skills/skills` 디렉토리를 확인하여 관련 스킬이 있는지 찾습니다.
    *   *예시*: "SEO" 관련 요청 -> `seo-fundamentals` 또는 `content-creator` 확인.
    *   *예시*: "데이터베이스" 관련 요청 -> `database-design` 확인.
3.  **스킬 호출**: 스킬이 적용될 가능성이 1%라도 있다면, 해당 스킬 디렉토리의 `SKILL.md`를 `view_file`로 읽습니다.
4.  **프로토콜 준수**: 스킬의 지침을 정확히 따릅니다.
5.  **스킬 없음?**: 적용할 스킬이 없다면, 표준 `concise-planning` 또는 `clean-code` 원칙을 따릅니다.

> **규칙**: 구체적인 스킬이 존재한다면 절대로 "어떻게 하는지 안다"고 단정하지 마십시오. 최신 프로젝트 표준을 따르기 위해 반드시 스킬 문서를 읽어야 합니다.
